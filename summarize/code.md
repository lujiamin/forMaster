## 天勤模拟卷8套
#### 一
整型一维数组，有正数有负数，求连续的一个或多个整数组成的子数组，使其和最大.  
算法思想：遍历数组，用事先定义好的求和变量，加上当前元素得到一个新的和，先判断这个和是否比前面已经记录的最大子数组和大，如果大，则更新此记录。然后判断这个和是否为负数，如果为负数，将这个和重置为0.  
```
void findmaxSum(int a[],int n){
    int sum;
    int max;
    int i;
    max = a[0];
    sum = 0;
    for(i=0;i<n;i++){
        sum += a[i];
        if(sum > max){
            max = sum;
        }
        if(sum < 0){
            sum = 0;
        }
    }
    printf("%d\n",max);
}
```
